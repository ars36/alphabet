# How to Use

Install SGA-Regular.ttf 

# To use as System language in Windows

!! Very Dangerous !!
!! Do not use unless you know what you are doing !!

Run vorticon.reg to set SGA-Regular as system language,
and vorticon-restore.reg to return it to normal.

May need to restart Windows for it to take effect.
